## ðŸš€ Getting Started

### For Authorized Participants

```bash
# Clone the repository
git clone https://github.com/GCagent/nobleport.etf.git
cd nobleport.etf

# Install dependencies
npm install

# Configure credentials
cp .env.example .env
# Add your AP credentials and blockchain keys
```
